## 2. Теоретическое введение

Основными функциональными элементами любой ЭВМ являются центральный
процессор, память и периферийные устройства.\
Центральный процессор выполняет обработку информации и координацию всех
узлов.\
В его состав входят арифметико-логическое устройство (АЛУ), устройство
управления (УУ) и регистры.

------------------------------------------------------------------------

## 3. Процесс создания и обработки программы на языке ассемблера

1.  **Создание каталога и файла программы:**

    ``` bash
    mkdir -p ~/work/arch-pc/lab04
    cd ~/work/arch-pc/lab04
    nano hello.asm
    ```

2.  **Вводим следующий код программы:**

    ``` asm
    section .data
        msg db 'Hello, world!', 0Ah

    section .text
        global _start

    _start:
        mov eax, 4
        mov ebx, 1
        mov ecx, msg
        mov edx, 13
        int 0x80

        mov eax, 1
        mov ebx, 0
        int 0x80
    ```

------------------------------------------------------------------------

## 4. Компиляция и линковка

После сохранения файла необходимо выполнить трансляцию программы в
объектный код:

``` bash
nasm -f elf hello.asm
```

Если ошибок нет, выполняем компоновку:

``` bash
ld -m elf_i386 hello.o -o hello
```

После этого проверяем наличие исполняемого файла с помощью команды `ls`
и запускаем программу:

``` bash
./hello
```

------------------------------------------------------------------------

## 5. Расширенный синтаксис командной строки NASM

Полный вариант команды `nasm` может содержать дополнительные параметры:

``` bash
nasm -o obj.o -f elf -g -l list.lst hello.asm
```

**Описание параметров:** - `-o` --- задаёт имя объектного файла\
- `-f` --- задаёт формат (elf)\
- `-g` --- включает отладочные символы\
- `-l` --- создаёт файл листинга

------------------------------------------------------------------------

## 6. Запуск программы

Исполняемый файл запускается командой:

``` bash
./hello
```

Программа должна вывести на экран:

    Hello, world!
